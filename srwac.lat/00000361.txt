Post-ovi tagovani odzivan dizajn
Responzivan web dizajn predstavlja novi trend na polju web dizajna. Šta je zapravo to Responzivan web dizajn?
Sama reč Response, odnosno Responsive referiše u našem rečniku nečemu što je odgovarajuće, odnosno odzivno, onaj koji daje odgovor, koji odgovara na interakciju. Mogli bi da ga nazovemo i Odzivan ili još bolje Prilagodjiv web dizajn. O čemu se dakle tu radi. Svaki web sajt koji vidimo kao jednu kompaktnu celinu je zapravo slika sačinjenu od mnoštva sličica i kodova koje su odgovarajuće uklopljene u Html-u i CSS-u, alatima za kodiranje. Naravno, tu imamo i veliki broj skripti koje daju efekte i pokreću slike koje mi doživljavamo kao slajdove, uvećanja slika, padajuće menije, ili otvaranja različitih formi itd. Kodiranje predstavlja uklapanje sličicica od kojih je sastavljen web sajt. Međutim, problem sa kojim se suočavamo je što web sajt gledamo kroz prozor internet browsera (Internet Explorer, Google Chrome, Mozilla Firefox, Opera, Safari..). Šta se dešava kada poželimo da prozor u kojem je web sajt smanjimo ili ili suzimo? Sve te sličice koje su pravilno pozicionirane će ostati na svom mestu (ako je reč o tome da su apsolutno pozicionirane, odnosno skupljaće se do određene mere kada će početi da se preklapaju sličice sa tekstovima i ostalim elementima, ikonicama, forama itd - ukoliko je reč o relativnom pozicioniranju elemenata sajta). U oba slučaja sam layout web sajta ostaće isti, i u najviše slučajeva mi nećemo moći da imamo kompletnog uvida u ceo sadržaj web sajta u prozoru internet pretraživača ili njegove delove, ukoliko ne skrolujemo web sajt levo desno po horizontalnoj osi. U intezivnom razvoju mobilnih platformi (mobilnih telefona, tableta, padova, netbookova itd) ovaj problem je postao još očigledniji. Praktično je nemoguće videti kompletan sadržaj web sajta. Privremeno rešenje su pružile mobilne verzije web sajtova koje predstavljaju paralelne verzije sajtova koje su rađene. Međutim i one imaju svoje granice u prikazu sadržaja jer se rade za unapred određene standardizovane rezolucije.
Responzivan web dizajn tu unosi jednu novinu. On praktično omogućuje web sajtu da se njegov sadržaj automatski prilagodi korisničkoj platformi. Web sajtovi rađeni na bazi responzivnog dizajna će biti automatski prilagođeni korisničkom prikaz na ekranu. Sada više nema veze da li ćete vaš web sajt pregledati sa vašeg telefona, tableta, netbooka ili monitora - prikaz layouta će vam uvek biti prilagođen vašem browseru. Kompletan sadržaj će se prilagoditi prikazu vašeg pretraživača i vi ćete moći da neometano pregledate sve sadržaje bez potrebe da skrolujete levo desno ili zumirate sadržaj kako bi mogli da pregledate sve sadržaje web sajta. Responzivan web dizajn u tom smislu predstavlja korak napred u prilagodjavanju sadržaja web sajta i layouta web sajta prema korisniku.
U pitanju je web sajt koji se bavi tematikom web dizajna i web developmenta, aktuelnim alatima i programerskim tehnikama. Na prvi pogled primetićete da web sajt ima jako puno sadržaja koji su pozicionirani u okviru jednog određenog layouta-rasporeda. Sada uhvatite ivicu vašeg browsera mišem i pokušajte da smanjite prozor browsera. Šta se dešava? - Web sajt automatski kreće da menja svoj raspored zadržavajući pri tom navigaciju i pristup sadržajima po istom principu kao što je to bilo kada ste pristupili web sajtu Probajte da smanjite veličinu browsera na najmanju moguću meru - sada tražite model prikaza kakav bi bio na mobilnim telefonima. Šta se dešava? Web sajt automatski prilagođava sadržaj veličini prozora vašeg browsera prikazujući neometano sve sadržaje, bez preklapanja tekstova formi i ostalih elemenata Responzivan web dizajn ovde konkretno pokazuje svoju pravu veličinu i značaj. Dostupnost sadržaja omogući će poseticima vašeg web sajta jedno potpuno novo korisničko iskustvo, jedan svet u kojem mogu da se osećaju komforno i lako bez obzira kako i sa čim pristupaju vašem web sajtu.
