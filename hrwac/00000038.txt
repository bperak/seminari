Poduzeće Korund d. o. o. bavi se širokim spektrom djelatnosti: od izrade i dizajna novih web stranica kao i održavanja postojećih web stranica, registracija svih vrsta domena, optimizacije web stranica za sve vodeče internetske tražilice kao što su Google, Yahoo ili MSN pa do usluge web hostinga. 
JOOMLA 
Joomla CMS 
Kod izrade web stranica služimo se Joomla sustavom upravljanja sadržajem (Joomla CMS) što nam omogućuje ponuditi vrlo povoljne cijene našim klijentima. Pogodnosti Joomla CMS sustava su trajna mogućnost nadopunjavanja stranice novim sadržajima koji možete obavljati sami na vrlo jednostavan način od trenutka objavljivanja stranice. Broj i vrsta promjena koje želite unijeti je neograničen. 
CSS i XHTML 
CSS 
U dizajnu i izradi web stranica koristimo se i CSS i XHTML programskim jezicima. CSS ili Cascading Style Sheet omogučuje izradu i dizajn web stranica bez upotrebe tablica. CSS služi da bi se sadržaj web stranice odvojio od samog dizajna stranice, što omogućuje brže učitavanje stranice u svim pretraživačima. 
Optimizacija za Tražilice SEO 
Naša specijalnost je i optimizacija web stranica za tražilice ili SEO (Search engine optimization). Ukoliko niste zadovoljni položajem koje zauzima vaša web stranica na glavnim tražilicama kao što su Google, Yahoo ili MSN Live obratite nam se i zajedno če mo pronaći rješenje koje će udovoljiti vašim zamislima. 
Neke od naših specijalnosti su paketi za trgovce rabljenim automobilima, oni namijenjeni poduzetnicima koji se bave prodajom i najmom nekretnina, paketi koji pokrivaju turističku ponudu od najma soba, apartmana i kuća za odmor, njihove prodaje i šire. 
Ukoliko želite biti prisutni na webu te za vas ili vašu tvrtku želite naručiti izradu web stranice, molimo, kontaktirajte nas.
